## Project Outline:
### Project Name: 
> 단체 톡방 채팅점유율, 채팅온도 분석 서비스
### Implement Goal: 
> SNS 대화 데이터를 분석, 채팅방의 온도와 인물별 채팅 점유율을 분석하여 대시보드화하여 출력
>> * 채팅 온도란?: 채팅방의 분위기를 수치화한 만들어낸 단위로 대화의 내용이 긍정적일수록 높고, 부정적일수록 낮다.
### Why?: 
> 채팅방 특성을 파악하고 시각화를 진행함으로서 관리의 효율성 증가 기대
